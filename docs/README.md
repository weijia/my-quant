# 量化交易工具功能说明

本文档介绍策略表格中的五个核心功能列：条件配置、条件单、快捷、高级设置、高级快捷。

---

## 功能总览

| 功能列 | 类型 | 数量 | 说明 |
|--------|------|------|------|
| 条件配置 | 配置列 | 4个输入框 | 设置交易金额、数量、价格、成交额 |
| 条件单 | 操作列 | 6个按钮 | 基于额/量，1%涨跌幅度设定买入卖出条件单 |
| 快捷 | 操作列 | 3个按钮 | 快速发送单条条件单（固定0.5%） |
| 高级设置 | 配置列 | 4个输入框 | 设置上涨/下跌趋势的买卖百分比 |
| 高级快捷 | 操作列 | 2个按钮 | 发送双向条件单（使用高级设置的百分比，按金额） |

---

## 1. 条件配置

### 1.1 功能说明

条件配置是一个配置列，用于设置交易金额、数量、价格和查看成交额。这些值会被「快捷」、「条件单」和「高级快捷」功能使用。

### 1.2 界面布局

```
条件配置
┌──────────────────┐
│ 额: [20000]     │
│ 量: [1400]      │
│ 价: [10.50]     │
│ 总: 14700       │
│ [1/4] [1/2]     │
└──────────────────┘
```

| 字段 | 含义 | 说明 |
|------|------|------|
| 额 | 交易金额 | 用于「条件单」和「高级快捷」按金额计算交易数量，缺省值为20000元 |
| 量 | 交易数量 | 用于「快捷」和「条件单」，缺省值为1/4持仓 |
| 价 | 当前价格 | 手动输入或自动获取，用于计算成交额和条件单数量 |
| 总 | 成交额 | 自动计算：量 × 价 |

### 1.3 数量计算规则

- **1/4持仓**：持仓数量 ÷ 4，向下取整到100的倍数
- **最低保底**：如果计算结果为0或小于100股，则最低设置为100股

---

## 2. 条件单

### 2.1 功能说明

条件单是一个操作列，包含6个按钮。基于条件配置中的「额」和「量」，以1%涨跌幅度设定买入卖出条件单。

### 2.2 按钮说明

| 按钮 | 类型 | 功能 | 说明 |
|------|------|------|------|
| 额↑买 | 金额 | 上涨1%买入 | 使用「额」计算数量，上涨1%时买入 |
| 额↓卖 | 金额 | 下跌1%卖出 | 使用「额」计算数量，下跌1%时卖出 |
| 额双向 | 金额 | 双向条件单 | 上涨1%买入 + 下跌1%卖出 |
| 量↑买 | 数量 | 上涨1%买入 | 使用「量」作为数量，上涨1%时买入 |
| 量↓卖 | 数量 | 下跌1%卖出 | 使用「量」作为数量，下跌1%时卖出 |
| 量双向 | 数量 | 双向条件单 | 上涨1%买入 + 下跌1%卖出 |

### 2.3 数量计算

- **基于金额**：数量 = 额 ÷ 价格（向下取整到100的倍数）
- **基于数量**：直接使用条件配置中的「量」值

---

## 3. 快捷

### 3.1 功能说明

快捷是一个操作列，包含三个快速下单按钮。用于快速发送单条条件单，百分比固定为0.5%。

### 3.2 按钮说明

| 按钮 | 功能 | 说明 |
|------|------|------|
| ↑买 | 上涨0.5%买入 | 发送一条条件单：当前价格上涨0.5%时买入 |
| ↓卖 | 下跌0.5%卖出 | 发送一条条件单：当前价格下跌0.5%时卖出 |
| 双向 | 同时买卖 | 发送两条条件单：上涨0.5%买入 + 下跌0.5%卖出 |

### 3.3 交易数量

快捷按钮使用「条件配置」中的量值，如果没有设置则使用1/4持仓数量。

---

## 4. 高级设置

### 4.1 功能说明

高级设置是一个配置列，用于设置上涨趋势和下跌趋势时的买卖百分比。这些百分比会被「高级快捷」功能使用。

### 4.2 界面布局

```
高级设置
┌─────────────────────┐
│ ↑ [买%] [卖%]       │
│ ↓ [卖%] [买%]       │
└─────────────────────┘
```

| 行 | 输入框 | 含义 |
|----|--------|------|
| 第一行（↑） | [买%] [卖%] | 上涨趋势时：上涨X%买入，下跌Y%卖出 |
| 第二行（↓） | [卖%] [买%] | 下跌趋势时：下跌X%卖出，上涨Y%买入 |

### 4.3 缺省值

| 趋势 | 买入% | 卖出% |
|------|-------|-------|
| 上涨趋势（↑） | 0.1% | 0.5% |
| 下跌趋势（↓） | 0.5% | 0.1% |

**设计逻辑**：上涨趋势时小比例买入（0.1%）、大比例止盈（0.5%）；下跌趋势时小比例止损（0.1%）、大比例抄底（0.5%）。

---

## 5. 高级快捷

### 5.1 功能说明

高级快捷是一个操作列，包含两个双向按钮。使用「高级设置」中的百分比，按「条件配置」中的金额计算数量。

### 5.2 按钮说明

| 按钮 | 发送条件单 | 使用 |
|------|-----------|------|
| ↑买 | 上涨买入 + 下跌卖出 | 高级设置的上涨买入% + 下跌卖出%，按金额计算数量 |
| ↓卖 | 下跌卖出 + 上涨买入 | 高级设置的下跌卖出% + 上涨买入%，按金额计算数量 |

### 5.3 数量计算

高级快捷按钮显示的数量 = 条件配置中的金额 ÷ 当前价格

### 5.4 与条件单的区别

| 区别 | 条件单 | 高级快捷 |
|------|--------|----------|
| 涨跌幅度 | 固定1% | 使用高级设置的百分比 |
| 数量计算 | 基于额或量 | 基于额 |
| 按钮数量 | 6个 | 2个 |

---

## 6. 功能协同

这些功能可以协同工作，形成完整的交易策略配置流程：

1. **步骤1**：在「条件配置」中设置交易金额（额）和交易数量（量）
2. **步骤2**：在「高级设置」中设置上涨/下跌趋势的买卖百分比
3. **步骤3**：使用「条件单」（1%固定幅度，可选额/量）或「高级快捷」（自定义幅度，按金额）发送条件单
4. **步骤4**：使用「快捷」进行快速单条下单（固定0.5%）

---

## 7. 使用场景示例

### 7.1 场景1：标准条件单

使用「条件单」的「额↑买」按钮：

- 系统会在价格上涨1%时自动买入（使用条件配置的金额计算数量）
- 适合按金额控制仓位的场景

### 7.2 场景2：固定数量交易

使用「条件单」的「量↑买」按钮：

- 系统会在价格上涨1%时自动买入（使用条件配置的数量）
- 适合按固定股数交易的场景

### 7.3 场景3：自定义百分比

使用「高级快捷」的「↑买」按钮：

- 系统会在价格上涨0.1%时自动买入（使用高级设置的百分比）
- 同时设置下跌0.5%时自动卖出（止盈止损）

---

## 8. 列显示设置

这些功能列可以通过「列选择」来显示或隐藏：

- 在表格右上角的设置菜单中可以找到「列选择」选项
- 勾选需要的列即可显示，不勾选则隐藏
- 每个用户的列选择设置会保存在浏览器本地
